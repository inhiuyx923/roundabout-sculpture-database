# Roundabout Sculpture Archive Public Site

This folder is the clean publishable version of the Roundabout Sculpture Archive prototype.

Upload this whole `public-site` folder to a static hosting service such as Netlify or Vercel.

Included:

- `index.html`: the website page.
- `data/roundabouts-enriched.js`: the public point database used by the map.

Not included:

- Original KMZ file.
- CSV exports.
- intermediate geocoding files.
- private working files.

Notes:

- The site uses Leaflet and OpenStreetMap map tiles from public CDNs.
- The data includes locations, editorial color categories, descriptions, and Google Maps links.
- It does not include stored Google Street View screenshots.

Live site: https://roundabout-sculpture-database.netlify.app/
